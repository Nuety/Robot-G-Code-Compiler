The robotic_arm is a URDF file and can be visualized using jupyter lab.
https://jupyterlab-urdf.readthedocs.io/en/latest/gen_getting_started.html

## Installation

Use pip to install jupyter lab and the URDF extension.

"pip install jupyterlab"
"pip install jupyterlab-urdf"

## Usage

Write the following in the terminal:

"jupyter lab"
